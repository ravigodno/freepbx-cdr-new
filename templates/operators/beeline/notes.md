# Билайн

- Уточняйте публичный SIP server и формат авторизации для конкретной услуги.
- Caller ID и DID могут требовать отдельного согласования.
- При 403 проверьте auth username, from user и разрешенный CID.
- При one-way audio проверьте NAT, RTP range и SIP ALG.
- Шаблоны являются отправной точкой и требуют проверки на конкретной АТС.
