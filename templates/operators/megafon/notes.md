# МегаФон

- Проверяйте требования к `fromDomain`, исходящему CID и регистрации.
- Разные корпоративные продукты могут использовать разные SIP server values.
- 403 обычно связан с авторизацией или Caller ID.
- 488 может указывать на DTMF, codec или media negotiation.
- Шаблоны не гарантируют работу без проверки на конкретной АТС.
