# PBXPuls AI Agent Rules

## Language

- Всегда отвечай пользователю на русском языке.
- Все объяснения, отчеты, планы работ, описания изменений и комментарии пиши только на русском.
- Код, команды Linux, TypeScript, JavaScript, SQL и API оставляй на языке оригинала.
- Commit messages оставляй на английском.
- Не переключайся на английский без явной просьбы пользователя.

## Development

- Никогда не ломай обратную совместимость.
- Перед изменениями сначала анализируй проект.
- После каждого изменения запускай npm run build.
- Если сборка не проходит — исправляй автоматически.
- Перед завершением показывай git diff.
- Если изменение рискованное — сначала объясни последствия.
